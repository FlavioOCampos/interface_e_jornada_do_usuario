# Projeto Final – IA Prototipada

## 📌 Descrição
Desenvolvimento de uma solução digital com uso de inteligência artificial, incluindo prototipação e validação da experiência do usuário.

## 🎯 Objetivo
- Criar uma solução inovadora utilizando IA
- Desenvolver protótipo completo
- Aplicar conceitos de UX/UI

## 🛠 Atividades
- Definição do problema
- Desenvolvimento da solução
- Criação de protótipo no Figma
- Testes de usabilidade
- Ajustes e melhorias

## 📎 Entregáveis
- Protótipo completo
- Descrição da solução
- Resultados dos testes

## 💡 Resultados Esperados
Solução funcional com boa experiência do usuário e aplicação de tecnologia.
