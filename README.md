## Sobre o Projeto

Este repositório apresenta a atividade de **Teste de Usabilidade (Exercício 01)** da disciplina **Interface e Jornada do Usuário**.

O objetivo é avaliar protótipos mobile desenvolvidos pelos estudantes e verificar se a interface permite que o usuário realize a tarefa principal com **clareza, facilidade e autonomia**.

---

## Objetivo da Atividade

A atividade busca:

- Avaliar a **usabilidade** das interfaces
- Identificar **problemas de navegação**
- Coletar **feedback dos usuários**
- Sugerir **melhorias na experiência do usuário**

## Metodologia

A avaliação é realizada entre colegas da turma:

1. Publicação do protótipo
2. Teste de **3 projetos de colegas**
3. Comentário **"TESTE"** antes da avaliação
4. Realização do teste de usabilidade
5. Preenchimento do formulário de avaliação

⚠ Cada projeto pode receber **no máximo 3 testes**.

---

## Critérios para Pontuação

Para que a atividade seja validada, o estudante deve:

✔ Testar **3 interfaces de colegas**  
✔ Preencher **3 formulários de avaliação**  
✔ Receber **pelo menos 3 avaliações** no próprio projeto
